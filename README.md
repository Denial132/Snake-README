# Snake Game

## Description
This is a classic Snake game created in Python using Pygame. The player controls a snake, collects apples to increase their score, and avoids colliding with walls or themselves.

## Features
- Main menu
- Snake movement
- Score and high score
- Pause menu
- Restart game
- Sound settings
- Speed settings
- Game Over screen
- Saved high score

## Controls

| Key | Action |
|------|--------|
| Arrow Keys | Move snake |
| P | Pause/Resume |
| Esc | Return to menu |

Buttons can also be used to navigate the menus.

## Requirements
- Python 3.x
- Pygame

Install Pygame with:

```bash
pip install pygame
```

## How to Run

Run the game with:

```bash
python Snake.py
```

## Files

- `Snake.py` – Main game
- `assets/` – Images, fonts and sounds (Make sure to put the background and the sound files into a file called assets)
- `save.json` – Stores the high score
