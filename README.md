# 🐍 Snake Game

A classic Snake game built with **Python and Pygame**. Control the snake, eat food, grow longer, and try to beat your high score!

## 🎮 Features

* Classic Snake gameplay
* Score and high score tracking
* Save system for progress
* Background music and sound effects
* Pause menu
* Main menu with buttons
* Settings menu:

  * Toggle sound on/off
  * Change game speed
* Multiple speed modes:

  * Slow
  * Normal
  * Fast
  * Extreme
* Turbo mode using the **F key**
* Grid-based movement and collision detection

## 🕹️ Controls

| Key        | Action           |
| ---------- | ---------------- |
| Arrow Keys | Move the snake   |
| P          | Pause/unpause    |
| F          | Turbo speed      |
| ESC        | Go back in menus |

## 📦 How to Play

### Windows

1. Download the latest release.
2. Run `Snake.exe`.
3. Start playing!

No Python or Pygame installation is required.

## 💾 Save System

The game automatically saves:

* High score
* Current score
* Snake position
* Food position
* Settings

Your save file (`save.json`) is created automatically when the game runs.
